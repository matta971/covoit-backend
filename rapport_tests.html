Avertissements (⚠️ WARN)

Spring Security retourne systématiquement 403 pour les requêtes sans token, là où les specs et les features attendent 401. Cela peut être corrigé en configurant HttpSecurity.exceptionHandling().authenticationEntryPoint() pour renvoyer 401.

Le refresh token renvoie le même accessToken si celui-ci n'a pas encore expiré. Comportement acceptable si intentionnel, sinon activer la rotation de token.
